## General Europe Travel Tips

- **Currency**: Most EU countries use the Euro (€). Czech Republic uses Czech Koruna (CZK). UK uses GBP.
- **Plug type**: Europe uses Type C/E/F plugs (2 round pins). Bring a universal adapter.
- **Health**: European Health Insurance Card (EHIC) provides basic coverage for EU citizens. All others should have private insurance.
- **Transport**: Eurail passes are cost-effective if visiting 3+ countries. Book high-speed trains in advance for best prices.
- **Tipping**: Generally 10% at restaurants in Western Europe. Less expected in Eastern Europe.
